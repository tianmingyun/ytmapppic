# 插图1：OAuth 房卡授权流程图

浓彩水粉画风格的流程图，主题为“会员卡与房卡的接力棒传递”。

画面分为5个连续场景，用柔和的箭头连接，采用丰富的水粉笔触和温暖明亮的色彩（金色、橙红、宝蓝、紫色）。

场景1（左侧）：一位用户手持黑色“SuperGrok 会员卡”，卡上写着“终身尊贵黑卡”。

场景2：会员卡变成一个蓝色链接按钮，链接指向 Twitter 登录页面。

场景3：用户在手机浏览器上点击“Authorize app”，画面有 Twitter 蓝色鸟图标。

场景4：浏览器地址栏显示 `code=ABC123...` 和一串长长的 `XAI_OAUTH_VERIFIER` 钥匙，钥匙形状醒目。

场景5（右侧）：钥匙插入一个发光的保险箱，箱内出现 “Access Token” 和 “auth.json” 文件，箱子上写着“永久激活”。

每个场景下方用清晰手写中文小字标注步骤：
1. 持有黑卡
2. 生成授权链接
3. Twitter 授权
4. 复制 Code + Verifier
5. 换取房卡并备份

整体风格：浓彩水粉画，厚重笔触，信息图风格但艺术化，色彩丰富，适合技术文档插图。
