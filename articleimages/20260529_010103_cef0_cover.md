# 微信公众号封面图 Prompt

**主题**：三类人关系边界 — 核心人、关系人、路人

**构图**（浓彩水粉画风格）：
- 画面采用 cinematic 2.35:1 横幅构图，适合微信公众号封面
- 浓彩水粉画质感：厚重笔触、丰富色彩叠加、可见画笔肌理、略带纸张纹理
- 色调以暖橙、深蓝、土褐为主，辅以鲜明的对比色，情绪内省而有力

**视觉元素**：
- 画面中央偏左：三条不同质地的“路”或“线”从不同方向延伸而来，在一个模糊的十字路口交汇
  - 最粗最实的那条路（暖金橙色）：代表「核心人」，有温暖光晕
  - 中等虚实相间的路（冷灰蓝）：代表「关系人」，边界清晰
  - 最淡最细的路（冷灰）：代表「路人」，逐渐消散
- 路口处站着一个简化的背影人影（非写实），低头思考，身上有三条不同粗细的线延伸出去
- 背景是抽象的城市与自然交界的水粉渲染，色块丰富但不凌乱
- 右上角留白较多，适合叠加标题

**文字**：
- 主标题使用干净而有力的手写体或衬线体，颜色为深褐或深蓝，与画面融合
- 标题位置：画面右侧或下方黄金分割处
- 副标题小字置于主标题下方

**整体氛围**：
- 不是戏剧化冲突，而是安静的顿悟感
- 光线柔和但有方向性，像黄昏或清晨的暖光
- 色彩浓郁但不艳俗，体现“累”与“清醒”的双重情绪

**禁止**：
- 不要出现真实的人脸
- 不要出现手机、电脑等现代元素
- 不要把三种人画成卡通人物直接贴标签

**参考色**：厚重水粉的橙红、群青、赭石、熟褐、象牙黑的丰富叠加
