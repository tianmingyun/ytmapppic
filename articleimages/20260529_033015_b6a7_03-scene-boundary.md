# 插图 Prompt - 边界金句

**风格**：浓彩水粉画，情绪强烈，笔触厚重。

**构图**：
- 画面分为两个明显却又融合的场景（水粉自然过渡）
- 左侧（过去）：一张老旧的家庭合影照片被撕裂，照片中的人物用暖色渲染，但照片边缘已经泛黄破损
- 右侧（现在）：一个成年人的背影独自站在一个十字路口，路口有三条不同方向的路
- 十字路口处有一道明显但不刺眼的“界限”光（像用画刀刮出的高光）
- 整体色调从左侧的暖黄逐渐过渡到右侧的冷蓝灰，再在路口处重新出现一点暖光

**象征元素**：
- 被撕裂的照片代表“血缘不是免死金牌”
- 十字路口的三条路呼应三类人
- 界限光代表“交换才是成年人的通行证”

**情绪**：
- 不是悲伤，而是清醒后的坚定
- 有重量感，但最终有光

**色彩**：
- 浓郁水粉叠加：赭石、熟褐、群青、橙红、冷灰
- 撕裂处用强烈的对比色表现情感的断裂

**文字**（可选，融入画面）：
- 金句小字可以轻柔地出现在右侧路口处

**禁止**：
- 不要画真实人脸
- 不要出现手机、微信等元素
- 保持艺术性和象征性

**氛围**：一幅有哲学深度的浓彩水粉画。
