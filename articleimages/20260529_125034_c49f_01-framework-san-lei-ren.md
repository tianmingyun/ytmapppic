# 插图 Prompt - 三类人框架

**风格**：浓彩水粉画（thick gouache），笔触厚重、色彩丰富叠加、可见画刀和画笔肌理，纸张纹理明显。

**构图**：
- 16:9 横幅
- 画面左侧：一个简化的侧面人影（背对或侧对），低头思考
- 从这个人影身上延伸出三条不同质感的“关系线”，向右延伸到三个不同的终点
  - **核心人（自己人）**：最粗、最实、最温暖的橙金色实线，线条有光晕，像被温暖的光包裹
  - **关系人（合伙人）**：中等粗细、虚实相间的冷蓝灰线，边界清晰，像用尺子画出的
  - **路人（过客）**：最细、最淡的灰线，逐渐消散在空气中
- 三条线最终通向三个不同的抽象“门”或“终点”：
  - 左边温暖的家门（橙）
  - 中间清晰的办公室门（蓝）
  - 右边逐渐模糊消失的路（灰）
- 背景是抽象的城市与自然交界的水粉渲染，色块浓郁但和谐

**色彩**：
- 主色调：暖橙、深赭石、群青、熟褐、象牙黑
- 强调色：少量亮黄和冷灰形成对比
- 整体情绪：内省、清醒、有重量感

**文字**（可选，画面右下角小字）：
- 「核心人」「关系人」「路人」三个小标签，用手写体融入画中

**禁止**：
- 不要画真实的人脸
- 不要出现文字过多
- 不要卡通化

**氛围**：安静的顿悟，像一幅有思考深度的水粉画。
